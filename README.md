**Welcome! Today I will be showing you two different code files, one meant for an Arduino UNO, and one for your own computer which in this case is a Python script!**

You can download these files and review them at any time you would like to, and you can contact me for any updates you would need.

These files are meant to be used on a solar panels voltage mainly, I did this whole thing as a project to build my own LED solar panel on a breadboard with an Arduino UNO! Don't forget to connect your Arduino UNO to your computer before running the script, as otherwise the computer wont recognize what to read!

I originally planned to showcase the 3V breadboard solar panel at the school science fair, but I decided to not and instead make it open source for anyone to use!

Thank you for reading!
